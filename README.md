# 🌍 Global Air Pollution Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes global air pollution patterns and their impact on public health.

## Features

- Air Quality Index (AQI) Analysis
- Pollution Alert Monitoring
- Industry Impact Assessment
- Health Risk Evaluation
- Interactive Filters
- KPI Tracking

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling


## Key Insights

- Identified regions with high pollution levels.
- Analyzed health risks associated with poor air quality.
- Explored industrial influence on AQI trends.
